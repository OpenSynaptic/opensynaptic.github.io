## 📦 OpenSynaptic 是什么

| 维度 | 说明 |
| --- | --- |
| 核心目标 | 统一、压缩并分发异构传感网络的数据流 |
| 处理链路 | `sensors -> standardize(UCUM) -> compress(Base62) -> fuse(FULL/DIFF) -> dispatch(transport)` |
| 关键能力 | Python/Rust 双后端、可插拔传输器、ID 租赁、TUI + Web UI |
| 适用场景 | 传感网络、边缘网关、协议桥接、异构设备管理 |
