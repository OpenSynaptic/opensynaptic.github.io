📦 What is OpenSynaptic?​

AspectDescription
Core Purpose2-N-2 IoT protocol stack for standardizing, compressing, and dispatching sensor data across heterogeneous networks
Pipelinesensors → standardize (UCUM) → compress (Base62) → fuse (FULL/DIFF) → dispatch (transport)
Key FeaturesMulti-language backends (Python, Rust), pluggable transporters (UDP, MQTT, UART), device ID leasing, TUI + Web UI
ScalabilityTested on 1M+ messages, 12 protocol types, stress tested up to 200 concurrent workers
Use CasesSensor networks, edge computing, IoT gateways, protocol bridging, heterogeneous device management

# Install and run
pip install -e .
os-node demo --open-browser

# Run tests
py -3 -m pytest --cov=opensynaptic tests
python scripts/integration_test.py

# Plugin development
python -u src/main.py plugin-test --suite component
python -u src/main.py plugin-test --suite integration

# System information
python -u src/main.py native-check
python -u src/main.py core --set rscore --persist
