# 产品文档结构

## 📊 现有产品级别

本项目现在包含 **3 个同等级的独立产品**：

### 1. 🔷 OpenSynaptic (主产品)
- **文件位置**: `docs/opensynaptic/`
- **网页URL**: `https://opensynaptic.github.io/docs/`
- **导航栏标签**: "Docs"
- **侧边栏配置**: `sidebars.ts`
- **包含内容**:
  - `en_GB/` - English documentation
  - `zh_CN/` - 中文文档
  - API 参考
  - 架构文档
  - 功能指南
  - 报告和发布说明

### 2. ⚡ OSynaptic-FX (新产品)
- **文件位置**: `docs/osynaptic-fx/`
- **网页URL**: `https://opensynaptic.github.io/docs/osynaptic-fx/`
- **导航栏标签**: "OSynaptic-FX"
- **侧边栏配置**: `sidebars.osynaptic-fx.ts`
- **创建文件**: 在 `docs/osynaptic-fx/` 下创建 markdown 文件

### 3. 🌐 Nexa (新产品)
- **文件位置**: `docs/nexa/`
- **网页URL**: `https://opensynaptic.github.io/docs/nexa/`
- **导航栏标签**: "Nexa"
- **侧边栏配置**: `sidebars.nexa.ts`
- **创建文件**: 在 `docs/nexa/` 下创建 markdown 文件

### 4. 📚 Extensions (扩展插件)
- **文件位置**: `docs/opensynaptic/extensions/`
- **网页URL**: `https://opensynaptic.github.io/docs/extensions/`
- **导航栏标签**: "Extensions"
- **侧边栏配置**: `sidebars.extensions.ts`

### 5. 🎯 Playbooks (剧本/教程)
- **文件位置**: `docs/opensynaptic/playbooks/`
- **网页URL**: `https://opensynaptic.github.io/docs/playbooks/`
- **导航栏标签**: "Playbooks"
- **侧边栏配置**: `sidebars.playbooks.ts`

## 🚀 快速开始

### 为 OSynaptic-FX 添加文档

1. 在 `docs/osynaptic-fx/` 下创建文件：
```
docs/osynaptic-fx/
├── intro.md
├── installation.md
├── guides/
│   ├── getting-started.md
│   └── advanced-usage.md
└── api/
    └── reference.md
```

2. 更新 `sidebars.osynaptic-fx.ts` (可选，配置已自动扫描)

3. 运行构建：
```bash
npm run build
```

### 为 Nexa 添加文档

流程完全相同，只需在 `docs/nexa/` 下创建文件即可。

## 🔧 配置文件

- **主配置**: `docusaurus.config.ts` - 定义所有产品的路由和侧边栏
- **OpenSynaptic 侧边栏**: `sidebars.ts`
- **OSynaptic-FX 侧边栏**: `sidebars.osynaptic-fx.ts`
- **Nexa 侧边栏**: `sidebars.nexa.ts`
- **Extensions 侧边栏**: `sidebars.extensions.ts`
- **Playbooks 侧边栏**: `sidebars.playbooks.ts`

---

## 📝 注意事项

- 所有文件使用 Markdown (`.md`) 或 MDX (`.mdx`) 格式
- 侧边栏配置使用 `autogenerated` 模式，自动扫描目录下的所有文件
- 国际化 (i18n) 配置：支持英文 (en) 和中文 (zh-CN)
- 导航栏在 `docusaurus.config.ts` 的 `navbar.items` 中配置
