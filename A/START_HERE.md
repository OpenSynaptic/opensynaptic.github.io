# 🎉 从这里开始!

## ⚡ 5 分钟快速开始

### 1️⃣ 推送代码到 GitHub

```powershell
cd E:\opensynaptic.github.io
git add .
git commit -m "feat: 初始化 Docusaurus 文档网站"
git push origin main
```

### 2️⃣ 配置 GitHub Pages

访问: https://github.com/opensynaptic/opensynaptic.github.io/settings/pages

设置:
- Source: "Deploy from a branch"  
- Branch: `gh-pages`
- Folder: `/ (root)`

### 3️⃣ 等待部署完成 (2-5 分钟)

访问: https://opensynaptic.github.io 🎉
