жҳҫзӨә API е®һзҺ° - README

вң… е®Ңж•ҙе®һзҺ°вҖӢ

OpenSynaptic зҡ„иҮӘеҸ‘зҺ°еҸҜи§ҶеҢ–зі»з»ҹпјҢе…Ғи®ёжҸ’д»¶дҪҝз”Ёж ҮеҮҶ API жіЁеҶҢиҮӘе®ҡд№үжҳҫзӨәйғЁеҲҶпјҢз”ұ web_user е’Ң tui иҮӘеҠЁеҸ‘зҺ°е’Ңе‘ҲзҺ°гҖӮ

рҹҺҜ жӮЁиҺ·еҫ—дәҶд»Җд№ҲвҖӢ

ж ёеҝғзі»з»ҹпјҲ3 дёӘж–Үд»¶пјҢ1050+ иЎҢпјүвҖӢ

  • display_api.py - е…·жңү DisplayProvider & DisplayRegistry зҡ„ж ҮеҮҶжҳҫзӨә API
  • example_display_plugin.py - еҸӮиҖғзӨәдҫӢпјҲ4 дёӘжҸҗдҫӣзЁӢеәҸпјү
  • id_allocator_display_example.py - з”ҹдә§зӨәдҫӢпјҲ2 дёӘжҸҗдҫӣзЁӢеәҸпјү

йӣҶжҲҗпјҲдҝ®ж”№ 3 дёӘж–Үд»¶пјҢ165+ иЎҢпјүвҖӢ

  • tui/main.py - иҮӘеҠЁеҸ‘зҺ°е№¶жҳҫзӨәжҸҗдҫӣзЁӢеәҸ
  • web_user/main.py - еӨ„зҗҶжҸҗдҫӣзЁӢеәҸиҜ·жұӮ
  • web_user/handlers.py - 3 дёӘж–°зҡ„ HTTP API з«ҜзӮ№

ж–ҮжЎЈпјҲ5 дёӘж–Үд»¶пјҢ1900+ иЎҢпјүвҖӢ

  • DISPLAY_API_QUICKSTART.md - 5 еҲҶй’ҹеҝ«йҖҹејҖе§Ӣ
  • DISPLAY_API_GUIDE.md - е®Ңж•ҙеҸӮиҖғ
  • DISPLAY_API_IMPLEMENTATION_SUMMARY.md - жҠҖжңҜз»ҶиҠӮ
  • IMPLEMENTATION_COMPLETE.md - жү§иЎҢж‘ҳиҰҒ
  • DISPLAY_API_INDEX.md - еҜјиҲӘжҢҮеҚ—

рҹҡҖ еҝ«йҖҹејҖе§ӢпјҲ3 жӯҘпјүвҖӢ

з¬¬ 1 жӯҘпјҡйҳ…иҜ»еҝ«йҖҹејҖе§ӢпјҲ5 еҲҶй’ҹпјүвҖӢ

cat DISPLAY_API_QUICKSTART.md

з¬¬ 2 жӯҘпјҡжҹҘзңӢзӨәдҫӢпјҲ10 еҲҶй’ҹпјүвҖӢ

cat src/opensynaptic/services/example_display_plugin.py

з¬¬ 3 жӯҘпјҡеҲӣе»әжӮЁзҡ„жҸҗдҫӣзЁӢеәҸпјҲ15 еҲҶй’ҹпјүвҖӢ

from opensynaptic.services.display_api import DisplayProvider, register_display_provider

class MyDisplay(DisplayProvider):
    def __init__(self):
        super().__init__('my_plugin', 'metrics', 'My Metrics')
    
    def extract_data(self, node=None, **kwargs):
        return {'value': 42}

def auto_load(config=None):
    register_display_provider(MyDisplay())
    return True

рҹҺҜ дё»иҰҒеҠҹиғҪвҖӢ

вң… еҹәдәҺжҸ’д»¶ - ж ёеҝғд»Јз Ғдёӯж— зЎ¬зј–з Ғ
вң… иҮӘеҠЁеҸ‘зҺ° - жҸ’д»¶жіЁеҶҢпјҢзі»з»ҹиҮӘеҠЁеҸ‘зҺ°
вң… еӨҡз§Қж јејҸ - JSONгҖҒHTMLгҖҒTEXTгҖҒTABLEгҖҒTREE
вң… еҗ‘еҗҺе…је®№ - 100% е…је®№
вң… зәҝзЁӢе®үе…Ё - RLock дҝқжҠӨзҡ„жіЁеҶҢиЎЁ
вң… з”ҹдә§е°ұз»Ә - е·ІжөӢиҜ•е№¶йӘҢиҜҒ
вң… ж–ҮжЎЈе®Ңе–„ - 1900+ иЎҢж–ҮжЎЈ

рҹ“– ејҖе§Ӣйҳ…иҜ»вҖӢ

  1. DISPLAY_API_QUICKSTART.md вҶҗ д»ҺиҝҷйҮҢејҖе§Ӣ
  2. example_display_plugin.py вҶҗ жҹҘзңӢзӨәдҫӢ
  3. DISPLAY_API_GUIDE.md вҶҗ е®Ңж•ҙеҸӮиҖғ
  4. id_allocator_display_example.py вҶҗ зҺ°е®һзӨәдҫӢ

вңЁ ж”№еҸҳдәҶд»Җд№ҲвҖӢ

д№ӢеүҚпјҲзЎ¬зј–з ҒпјүвҖӢ

# еңЁ tui/main.py дёӯ - еҝ…йЎ»дҝ®ж”№ж ёеҝғд»Јз Ғ
def _section_custom(self):
    return {'metric': get_value()}

_SECTION_METHODS = {'custom': '_section_custom'}

д№ӢеҗҺпјҲеҹәдәҺжҸ’д»¶пјүвҖӢ

# еңЁд»»дҪ•жҸ’д»¶дёӯ - ж— йңҖдҝ®ж”№ж ёеҝғд»Јз Ғ
class CustomDisplay(DisplayProvider):
    def extract_data(self, node=None, **kwargs):
        return {'metric': get_value()}

def auto_load(config=None):
    register_display_provider(CustomDisplay())
    return True
