## 翻译质量标准

所有翻译遵循以下标准：

| 标准 | 要求 |
|------|------|
| **精确性** | 中文与英文对应率 ≥ 80%（表格、代码 100%）|
| **术语一致性** | 相同术语统一翻译，维护术语表 |
| **代码保留** | 所有代码块、命令、JSON 完全保留，不翻译 |
| **格式保留** | Markdown 格式完全保留，包括链接、表格、代码块 |
| **易读性** | 优先自然流畅的中文表达，而非机械直译 |
