快速文档查找

使用此页面快速跳转到正确的文档集合。

运行时首次启动​

使用以下命令之一直接启动 main：

# Windows 快捷命令（无需 Activate.ps1）
.\run-main.cmd run --once --quiet

# 已安装的入口点
os-node run --once --quiet

如果首次运行时缺少本机运行时库，启动会自动尝试本机构建一次，然后重试节点创建。
手动回退：

os-node native-check
os-node native-build
