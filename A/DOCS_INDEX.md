## 🔑 关键概念速览

### Docusaurus
- 现代静态网站生成器
- 基于 React 和 Markdown
- 易于自定义和扩展

### GitHub Pages
- 免费的静态网站托管
- 自动部署到 gh-pages 分支
- https://opensynaptic.github.io

### GitHub Actions
- 自动化构建和部署
- 配置文件: .github/workflows/deploy.yml
- 触发: push to main 分支

### 项目结构
```
docs/     → 文档页面
blog/     → 博客文章
src/      → 自定义组件
static/   → 静态资源
build/    → 构建输出
```
